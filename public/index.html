Is ZeroNet the future?

22 Jan, 2017 Technology Zeronet

What is ZeroNet? It’s a Decentralized web platform using Bitcoin cryptography and BitTorrent network. Which provides open, free & uncensorable websites. Why ZeroNet? Because it provides Peer-to-Peer connectivity without any central server and hence content is distributed directly to visitors. Uncensored because its nowhere to be found. No hosting cost as sites are served by visitors. Impossible to shutdown as its nowhere yet everywhere. No point of failure as the site will keep serving till the last peer serving.
